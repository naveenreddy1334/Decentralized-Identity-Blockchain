##  Getting Started

###  Prerequisites

Before getting started with Decentralized-Identity-Blockchain, ensure your runtime environment meets the following requirements:

- **Programming Language:** JavaScript
- **Package Manager:** Npm


###  Installation

Install Decentralized-Identity-Blockchain using one of the following methods:

**Build from source:**

1. Clone the Decentralized-Identity-Blockchain repository:
```sh
❯ git clone https://github.com/naveenreddy1334/Decentralized-Identity-Blockchain
```

2. Navigate to the project directory:
```sh
❯ cd Decentralized-Identity-Blockchain
```

3. Install the project dependencies:


**Using `npm`** &nbsp; [<img align="center" src="https://img.shields.io/badge/npm-CB3837.svg?style={badge_style}&logo=npm&logoColor=white" />](https://www.npmjs.com/)

```sh
❯ npm install
```




###  Usage
Run Decentralized-Identity-Blockchain using the following command:
**Using `npm`** &nbsp; [<img align="center" src="https://img.shields.io/badge/npm-CB3837.svg?style={badge_style}&logo=npm&logoColor=white" />](https://www.npmjs.com/)

```sh
❯ npm start
```
